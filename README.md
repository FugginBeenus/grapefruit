# grapefruit
A music library management app built with iPod Classic in mind. 

I noticed there were no apps that made switching from a streaming service like Apple Music or Spotify to using a music player like an iPod classic easy. 

Personally I still use Apple Music to have music when I don't bring my iPod, and also to discover new music, but I still want to keep my Plex library of music synced with my Apple Music library. My current workflow looks like this:
- Add a song to my Apple Music Library
- A while later I'll use Soulseek to find new songs in my Apple Music library
- Add the Soulseek results to my Plex library
- Plug in my iPod with Rockbox (for .flac) and use **Grapefruit** to update my playlists and library

Boom! All synced up. Takes maybe 5min. 

**THIS APP CAN**
- Import Apple Music playlists and match files with your local library to create the same playlist locally
- Use delta updating to only update what has changed in your library instead of having to sync your whole library.
